Convert python script to exe

Well you have made an amazing app using python. But it's not all. You have to make your python script an exe. Well I have recently made an app and successfully converted it to an exe. I will just share you how I did that. So guys, just continue reading!

What is an exe?

To start this you must know what is an exe. Well exe is a file extension. It is the short form of "executable". It contains icons, dlls(if you are windows user) etc. Im simple words, If we are talking about a python app, It will launch your app without any python launcher. You just have click on the icon and your app will be launched. To do that you have to follow some steps.

Step1:Install auto-py-to-exe

There are various types python libraries in pypi to convert .py to .exe. But the most easiest one is "auto-py-to-exe". It comes with a user friendly graphical interface and best for beginners. So to use that, open your tarminal(on windows command prompt or poweshell) and write pip install auto-py-to-exe and then press enter. You can see the picture below.

Step2: Run auto-py-to-exe

After finishing the installation of auto-py-to-exe, we have to run it. For this, we have to write in our tarminal auto-py-to-exe and press enter.

after that you will see a window like this.

Step3: Select your python script

When you will see the above page, you will see a box named "Script Location" for selecting your python script. Click the browse button beside it and then select your python script.

Step4: Select console window

This step is very important!. Below onefile, there is a section called console window. By default "console based" is selected in this section. You just have change it to "window based" by clicking the button right next to it (it is shown in the image below). If you have selected the "console based" option, everytime a console window would appear after running the exe and it doesn't look so good! that's why we selected the "window based option".

Step5: Select Icon

Now we need an icon. Without an icon no one will like your app. Under console window, There is a dropdown named "Icon". Click on the dropdown and a box will open. Click on the browse button and select your icon.
Note that: Your icon should be a .ico file. If your icon is not a .ico file, just convert it to .ico. You'll find some sites below to convert image to .ico Here are some sites to convert images to ico:-

  • icoconvert.com
  • Convertimage.net

Step6: Add additional files(if you have any)

Now an could contain some additional or external files like an image ,music(background sound), additional license text etc. If your app have any files like these you have to tell that to auto-py-to-exe. To do that, you will see an dropdown "Additional Files". click on it and you will see three option. One is for adding file, one is for adding folder and one is blank. Select your additional files by clicking one of these(it's your choice).

Step7: Convert!

Yay!! Our work is done. Now you just have to click on the Convert .py to .exe button and it will lead your scipt to glory! Open the output folder and search for your exe. The word "Application" will be next your exe.
