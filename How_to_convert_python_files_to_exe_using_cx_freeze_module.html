Convert python script to exe using cx_freeze

Hi guys! what's up. Welcome to Tricky Facts. In this article I am going to discuss how can you convert your python script to exe using cx_freeze. So, continue reading!

First you have to install the cx_freeze module. To do that, open your tarminal (command prompt or powershell in windows) and then write pip install cx_freeze and press enter like below.

after finishing the installation you have to create a python code. Open your editor and create a new python file and name it something. For now we are using "setup.py" and then see the code below.

  1. from cx_freeze import setup ,Executable
  2. setup(
  3. name="your exe name",
  4. version="version of your exe",
  5. description="description of your exe",
  6. options={"build_exe":{"packages" :["enter packages you have used in python script"],
  7. "include_files":["Add your additional files here(if you have any)"],
  8. "include_msvcr:True"}}
  9. executable=[Executable("your_python_script.py", icon="your exe icon path", base=base)]
  10. )

First from cx_freeze module, we have to import "setup" and "Executable". We did that in line 1.

in line 2, I've called setup(). Inside the parantheses or first brackets, I've written some options

  • name: Here you have to write the name of your exe or application
  • version: The version of your application or exe
  • options: This is the most important part! this contains some options for building the exe
  • includes: Here you have to write the name of the packages you have used in your main python script
  • include_files: If you have any external files in your main script like images or anything else, you have to write the path of the files in this option
  • include_msvcr: msvcr is the short form of "Microsoft Visual C++ Redistributable". You can add it in your exe in this option. It will already include the Redistributabe and user doesn't have to add it externally. It is a boolean value. If you want to add it,just set the value to True otherwise False
  • executable: Well in this option we have to write the path of our main script and the path of our application icon. You also have to set the base of your exe

Well, it's not all. There are a lot of commands and options in cx_frexe. Now I'll show you how can you can these by only one command line. Now, open your tarminal and write "setup.py --help" and then press enter. after that you will see something like this.
