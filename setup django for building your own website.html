Getting started with django!

Hi guys, welcome to tricky facts. If you are trying to make an website using python, then you are in the write place. Now we are going to learn how to get started with django in python. Hope you will like it.

Why django?

You all might be thinking that why I have chosen django for web development in python. Well the answer is really simple. Django is the best library for web developement in python and it is also written purely in python and it is also fast. It contains all the required tools for building a website. Some internet giants like google, youtube, dropbox even instagram are also created using django. So, that's why we are using django for web development. Now follow these steps to start working with django.

Step1: Install Visual studio code

To start working with django first you need install a code editor. I would suggest visual studio code.Because it is best for buiding any python projects and also has a built in tarminal to run codes. We will need that tarminal for django. you can also use pycharm. But in this tutorial I will be using visual sudio code .Here is the link to install them.

Step2: Install django

Before we can install django, first we need to create a new folder named "Project_Django". So just create this folder in any drive on your pc and then open visual studio code.If you are using visual studio code for the first time, at startup you will find line called "Open folder" just click on it and select the folder.

Now you can install the django library. Open the tarminal(Showed in image1) and write pip install django and then press enter. Django will be installed in your pc.
